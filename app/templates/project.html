PROJECT

CANCERAI — AI-Powered Cancer Detection Platform

A research-grade web application for breast cancer detection using a CNN+SVM hybrid deep learning model, trained on the BreakHis histopathology dataset, with integrated real-time doctor consultation.

Project Overview

CANCERAI is a comprehensive web platform designed to assist in the early detection of breast cancer using artificial intelligence. The system combines state-of-the-art deep learning techniques with classical machine learning to provide accurate and rapid analysis of histopathology images.

The platform serves two primary user roles: Patients who can upload histopathology images for AI analysis, and Doctors who can review AI-generated diagnoses, provide clinical assessments, and communicate with patients in real-time through an integrated chat system.

This project was developed as part of an academic initiative to demonstrate the practical application of AI in healthcare, with a focus on improving access to early cancer screening in underserved communities.
